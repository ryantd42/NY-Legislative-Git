# Prohibits agencies from charging a fee for records where an electronic copy is already available from a previous request made within the past six months

**Bill ID:** S2220
**Session:** 2023
**Sponsor:** James Skoufis
**Status:** Senate Floor Calendar

## Summary

Prohibits agencies from charging a fee for records where an electronic copy is already available from a previous request made within the past six months except for the actual cost of a storage device or media if provided to the requester.
