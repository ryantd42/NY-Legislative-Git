# Requires that any bill which provides revenue to the state in a non-recurring manner shall pass by a two-thirds majority in order to become law

**Bill ID:** A3037
**Session:** 2023
**Sponsor:** Edward Ra
**Status:** In Assembly Committee

## Summary

Requires that any bill which provides revenue to the state in a non-recurring manner shall pass by a two-thirds majority in order to become law.
