# Disallows an application for a charter school to be approved if a state monitor or fiscal monitor is not operating in such school district

**Bill ID:** A1986
**Session:** 2023
**Sponsor:** Kimberly Jean-Pierre
**Status:** In Assembly Committee

## Summary

Disallows an application for a charter school to be approved if a state monitor or fiscal monitor is not operating in such school district.
