# Provides for a license to sell liquor at retail for consumption on certain premises

**Bill ID:** S6741
**Session:** 2023
**Sponsor:** Michael Gianaris
**Status:** Signed by Governor

## Summary

Provides for a license to sell liquor at retail for consumption on certain premises located at Block 644, p/o Lot 1, Building 13, Queens county.
