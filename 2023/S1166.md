# Increases the penalty for rape in the first degree

**Bill ID:** S1166
**Session:** 2023
**Sponsor:** Peter Oberacker
**Status:** In Senate Committee
**PDF:** [S1166 PDF](https://legislation.nysenate.gov/pdf/bills/2023/S1166)

## Summary

Increases the penalty for rape in the first degree from a class B felony to an class A-II felony.
