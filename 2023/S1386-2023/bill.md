# Relates to requiring the metropolitan transportation authority to create ten-year capital program plans

**Bill ID:** S1386
**Session:** 2023
**Sponsor:** Leroy Comrie
**Status:** In Senate Committee
**PDF:** [S1386 PDF](https://legislation.nysenate.gov/pdf/bills/2023/S1386)

## Summary

Requires the metropolitan transportation authority to create ten-year capital program plans.
