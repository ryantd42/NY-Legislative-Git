# Relates to the establishment of a single computerized state DNA identification index and requires municipalities to expunge any DNA record stored in a municipal DNA identification index

**Bill ID:** A1877
**Session:** 2023
**Sponsor:** Stefani Zinerman
**Status:** In Assembly Committee

## Summary

Relates to the establishment of a single computerized state DNA identification index; requires municipalities to expunge any DNA record stored in a municipal DNA identification index.
