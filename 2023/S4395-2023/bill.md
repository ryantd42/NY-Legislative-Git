# Requires that prior to a contract with the state or other public entity being effective, a report must be filed with the comptroller disclosing financial interests

**Bill ID:** S4395
**Session:** 2023
**Sponsor:** Liz Krueger
**Status:** In Senate Committee

## Summary

Requires that prior to a contract with the state or other public entity being effective, a report must be filed with the comptroller disclosing financial interests.
