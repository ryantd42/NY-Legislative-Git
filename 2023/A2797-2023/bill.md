# Establishes the statewide kinship navigator program

**Bill ID:** A2797
**Session:** 2023
**Sponsor:** Andrew Hevesi
**Status:** In Assembly Committee

## Summary

Establishes the statewide kinship navigator program for kinship caregivers.
