# Relates to authorizing the Bellmore Fire District to file an application for exemption from real property taxes

**Bill ID:** S2622
**Session:** 2023
**Sponsor:** Steve Rhoads
**Status:** Senate Floor Calendar

## Summary

Makes clarifying changes regarding the authorization for the Bellmore Fire District to file an application for exemption from real property taxes.
