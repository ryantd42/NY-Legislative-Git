# Relates to the proper venue for an impounding organization to file a petition for a security

**Bill ID:** A1121
**Session:** 2023
**Sponsor:** Amy Paulin
**Status:** In Assembly Committee
**PDF:** [A1121 PDF](https://legislation.nysenate.gov/pdf/bills/2023/A1121)

## Summary

Relates to the proper venue for an impounding organization to file a petition for a security to be paid by the owner of a seized animal currently in the care of such organization.
