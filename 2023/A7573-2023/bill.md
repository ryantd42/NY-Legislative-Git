# Requires the state to pay real property taxes on the assessed value of the Moriah shock incarceration correctional facility

**Bill ID:** A7573
**Session:** 2023
**Sponsor:** Matthew Simpson
**Status:** In Assembly Committee

## Summary

Requires the state to pay all real property taxes on the assessed value of the Moriah shock incarceration correctional facility.
