# Mandates that every owner of a cabaret establishment, but which is not a restaurant shall install security cameras

**Bill ID:** S1429
**Session:** 2023
**Sponsor:** Jamaal Bailey
**Status:** In Senate Committee
**PDF:** [S1429 PDF](https://legislation.nysenate.gov/pdf/bills/2023/S1429)

## Summary

Enacts "Imette's Law"; mandates that every owner of a cabaret establishment, but which is not a restaurant, shall install security cameras at every entrance and exit of such establishment.
