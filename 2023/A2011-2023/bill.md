# Authorizes and directs the state board of elections to create an online petitioning system for designating petitions

**Bill ID:** A2011
**Session:** 2023
**Sponsor:** Robert C. Carroll
**Status:** In Assembly Committee

## Summary

Authorizes and directs the state board of elections to create an online petitioning system for designating petitions.
