# Provides an exemption to the prohibition on tinted windows in automobiles for emergency ambulance service vehicles

**Bill ID:** S1816
**Session:** 2023
**Sponsor:** Toby Ann Stavisky
**Status:** In Senate Committee
**PDF:** [S1816 PDF](https://legislation.nysenate.gov/pdf/bills/2023/S1816)

## Summary

Provides an exemption to the prohibition on tinted windows in automobiles for emergency ambulance service vehicles.
