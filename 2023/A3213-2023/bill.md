# Requires certain handicap accessible playground equipment in each playground within each state park

**Bill ID:** A3213
**Session:** 2023
**Sponsor:** Josh Jensen
**Status:** In Assembly Committee

## Summary

Requires at least one handicap accessible swing set in each state park.
