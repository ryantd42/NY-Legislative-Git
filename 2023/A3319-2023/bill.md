# Requires gas corporations and electric corporations in this state to file quarterly reports with the legislature regarding infrastructure upgrades

**Bill ID:** A3319
**Session:** 2023
**Sponsor:** Alicia Hyndman
**Status:** In Assembly Committee

## Summary

Requires gas corporations and electric corporations in this state to file quarterly reports with the legislature regarding infrastructure upgrades.
