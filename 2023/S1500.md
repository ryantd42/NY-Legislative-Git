# Allows the implementation of body cameras on police officers notwithstanding binding arbitration

**Bill ID:** S1500
**Session:** 2023
**Sponsor:** Michael Gianaris
**Status:** Stricken
**PDF:** [S1500 PDF](https://legislation.nysenate.gov/pdf/bills/2023/S1500)

## Summary

Allows the implementation of body cameras on police officers notwithstanding binding arbitration.
