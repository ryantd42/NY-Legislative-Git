# Relates to the development of a scannable code that links to translations of work zone and truss type construction signs

**Bill ID:** A2079
**Session:** 2023
**Sponsor:** Demond Meeks
**Status:** In Assembly Committee

## Summary

Relates to the development of a scannable code that links to translations of work zone and truss type construction signs.
