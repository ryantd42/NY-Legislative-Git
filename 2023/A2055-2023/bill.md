# Relates to the Long Island Rail Road weekend reduced fare program

**Bill ID:** A2055
**Session:** 2023
**Sponsor:** Khaleel Anderson
**Status:** In Assembly Committee

## Summary

Amends the Long Island Rail Road weekend reduced fare program to include the Far Rockaway station.
