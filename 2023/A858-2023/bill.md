# Relates to the state and municipal facilities program

**Bill ID:** A858
**Session:** 2023
**Sponsor:** Jonathan Jacobson
**Status:** In Assembly Committee
**PDF:** [A858 PDF](https://legislation.nysenate.gov/pdf/bills/2023/A858)

## Summary

Requires state and municipal facilities program grants be issued only with governor, senate or assembly sponsor.
