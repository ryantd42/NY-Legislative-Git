# Establishing the time period during which to file a claim for sexual harassment against the state in cases when a complaint was filed with a state or local agency

**Bill ID:** A1047
**Session:** 2023
**Sponsor:** Kenneth Zebrowski
**Status:** Stricken
**PDF:** [A1047 PDF](https://legislation.nysenate.gov/pdf/bills/2023/A1047)

## Summary

Establishes the time period during which to file a claim for sexual harassment against the state in cases when a complaint was filed with a state or local agency.
