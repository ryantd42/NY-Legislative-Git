# Establishes an economic grant program for the Second Avenue Subway construction project area

**Bill ID:** S808
**Session:** 2023
**Sponsor:** José M. Serrano
**Status:** In Senate Committee
**PDF:** [S808 PDF](https://legislation.nysenate.gov/pdf/bills/2023/S808)

## Summary

Establishes an economic development grant program for the Second Avenue Subway construction project area; defines micro business.
