# Extends the timeline for the return of security deposits for residential rental agreements from fourteen days to thirty days

**Bill ID:** A3278
**Session:** 2023
**Sponsor:** Nader Sayegh
**Status:** In Assembly Committee

## Summary

Extends the timeline for the return of security deposits for residential rental agreements from fourteen days to thirty days.
