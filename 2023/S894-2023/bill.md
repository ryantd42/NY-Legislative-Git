# Prohibits the issuance of a permit for the construction of a new multiple dwelling unless the multiple dwelling contains the minimum number of dwelling units required for the lot

**Bill ID:** S894
**Session:** 2023
**Sponsor:** Brad Hoylman-Sigal
**Status:** In Senate Committee
**PDF:** [S894 PDF](https://legislation.nysenate.gov/pdf/bills/2023/S894)

## Summary

Prohibits the issuance of a permit for the construction of a new multiple dwelling unless the multiple dwelling contains the minimum number of dwelling units required for the lot; defines minimum number of units required.
