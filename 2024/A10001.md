# Establishes the endocannabinoid system awareness program

**Bill ID:** A10001
**Session:** 2024
**Sponsor:** Anna Kelles
**Status:** In Assembly Committee
**PDF:** [A10001 PDF](https://legislation.nysenate.gov/pdf/bills/2024/A10001)

## Summary

Establishes the endocannabinoid system awareness program and work group.
