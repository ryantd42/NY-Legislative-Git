# Relates to state agency websites

**Bill ID:** S5410
**Session:** 2024
**Sponsor:** Nathalia Fernandez
**Status:** Signed by Governor
**PDF:** [S5410 PDF](https://legislation.nysenate.gov/pdf/bills/2024/S5410)

## Summary

Requires state agency websites to be mobile friendly.
