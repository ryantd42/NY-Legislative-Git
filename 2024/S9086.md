# Requires the commissioner create rules, regulations, policies, and procedures for the creation of a resilient electric vehicle charging system pilot program

**Bill ID:** S9086
**Session:** 2024
**Sponsor:** Kevin S. Parker
**Status:** In Senate Committee
**PDF:** [S9086 PDF](https://legislation.nysenate.gov/pdf/bills/2024/S9086)

## Summary

Requires the commissioner create rules, regulations, policies, and procedures for the creation of a resilient electric vehicle charging system pilot program to provide power through an extended outage.
