# Elevates assault of a code enforcement official to the class D felony of assault in the second degree

**Bill ID:** A6265
**Session:** 2024
**Sponsor:** Joe Angelino
**Status:** In Assembly Committee
**PDF:** [A6265 PDF](https://legislation.nysenate.gov/pdf/bills/2024/A6265)

## Summary

Elevates assault of a code enforcement official in the second degree to a class D felony.
