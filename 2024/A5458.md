# Relates to the common core state standards initiative

**Bill ID:** A5458
**Session:** 2024
**Sponsor:** Doug Smith
**Status:** In Assembly Committee
**PDF:** [A5458 PDF](https://legislation.nysenate.gov/pdf/bills/2024/A5458)

## Summary

Relates to the common core state standards initiative; establishes the department shall discontinue implementation of the common core state standards.
