# Includes assault on a corrections officer as a class C felony

**Bill ID:** A8221
**Session:** 2024
**Sponsor:** Sam Berger
**Status:** In Assembly Committee
**PDF:** [A8221 PDF](https://legislation.nysenate.gov/pdf/bills/2024/A8221)

## Summary

Includes assault on a corrections officer as a class C felony.
