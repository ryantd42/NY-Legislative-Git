# Requires windows in newly constructed multiple dwellings with two or more stories to be fitted with key-operated window locks

**Bill ID:** A8014
**Session:** 2024
**Sponsor:** Brian Cunningham
**Status:** In Assembly Committee
**PDF:** [A8014 PDF](https://legislation.nysenate.gov/pdf/bills/2024/A8014)

## Summary

Requires windows in newly constructed multiple dwellings with two or more stories to be fitted with key-operated window locks that restrict the window from opening more than four inches.
