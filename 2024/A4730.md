# Changes the official name of the Staten Island Expressway to the POW-MIA Memorial Highway

**Bill ID:** A4730
**Session:** 2024
**Sponsor:** Samuel Pirozzolo
**Status:** In Assembly Committee
**PDF:** [A4730 PDF](https://legislation.nysenate.gov/pdf/bills/2024/A4730)

## Summary

Changes the official name of the Staten Island Expressway to the "POW-MIA Memorial Highway".
