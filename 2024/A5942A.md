# Relates to the next generation 911 system, an emergency services internet protocol network

**Bill ID:** A5942A
**Session:** 2024
**Sponsor:** Kimberly Jean-Pierre
**Status:** In Assembly Committee
**PDF:** [A5942A PDF](https://legislation.nysenate.gov/pdf/bills/2024/A5942A)

## Summary

Requires the state shall offer a statewide emergency services internet protocol network (ESINet) available for connection to local public safety answering points (PSAPs).
