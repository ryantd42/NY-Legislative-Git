# Provides for state liability for potholes on state highways

**Bill ID:** S6144
**Session:** 2024
**Sponsor:** Timothy M. Kennedy
**Status:** Stricken
**PDF:** [S6144 PDF](https://legislation.nysenate.gov/pdf/bills/2024/S6144)

## Summary

Provides for liability of the state to any person who suffers damage as a result of a defect in a state highway.
