# Relates to swimming pool enclosures

**Bill ID:** S7731A
**Session:** 2024
**Sponsor:** Patrick M. Gallivan
**Status:** Senate Floor Calendar
**PDF:** [S7731A PDF](https://legislation.nysenate.gov/pdf/bills/2024/S7731A)

## Summary

Requires that release mechanism standards be developed by the state fire prevention and building code council based on internationally recognized standards.
