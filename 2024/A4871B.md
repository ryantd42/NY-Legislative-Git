# Requires certain state owned and operated parking facilities to install and maintain charging stations for plug-in electric vehicles

**Bill ID:** A4871BB
**Session:** 2024
**Sponsor:** Anna Kelles
**Status:** Vetoed
**PDF:** [A4871BB PDF](https://legislation.nysenate.gov/pdf/bills/2024/A4871BB)

## Summary

Requires certain state owned and operated parking facilities to install and maintain charging stations for plug-in electric vehicles.
