# Requires that for-hire vehicles have New York state issued license plates to receive a New York state inspection

**Bill ID:** S9532
**Session:** 2024
**Sponsor:** Leroy Comrie
**Status:** In Senate Committee
**PDF:** [S9532 PDF](https://legislation.nysenate.gov/pdf/bills/2024/S9532)

## Summary

Prohibits the New York state inspection of vehicles used for a for-hire service if such vehicles have out-of-state license plates.
