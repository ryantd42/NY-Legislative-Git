# Waives the state application fee for all civil service examinations for all veterans

**Bill ID:** A5761
**Session:** 2024
**Sponsor:** Alicia Hyndman
**Status:** In Assembly Committee
**PDF:** [A5761 PDF](https://legislation.nysenate.gov/pdf/bills/2024/A5761)

## Summary

Waives the state application fee for all civil service examinations for all veterans.
