# Provides for the creation of an online, searchable database to catalogue workforce development programs in the state

**Bill ID:** A6052
**Session:** 2024
**Sponsor:** Harry B. Bronson
**Status:** In Assembly Committee
**PDF:** [A6052 PDF](https://legislation.nysenate.gov/pdf/bills/2024/A6052)

## Summary

Provides for the creation of an online, searchable database to catalogue workforce development programs and funding available in the state.
